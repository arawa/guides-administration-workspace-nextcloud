# Guides d'administration des workspaces pour Nextcloud par Arawa

Réalisés pour l'application [Workspace](https://github.com/arawa/workspace/) développée par la société Arawa elle-même.

## Description

Ce dépôt contient des documentations destinées aux administrateur des workspace pour Nextcloud. Elles sont rédigées en français, au format ODT (éditées avec LibreOffice). Ces guides fonctionnels permettent de gérer les workspaces en tant que "General Manager" d'une part et en tant que "Workspace Manager" d'autre part.

# Fonctionnement des sources

Le document source est composé d'un document maître au format "odm" et de dépendances au format "odt". Chaque fichier "odt" est donc une brique, et toutes ces briques sont appelées dans un ordre particulier depuis le document maître pour former le document final. 

Si vous souhaitez personnaliser le guide, téléchargez l'ensemble des sources, puis modifiez le⋅s fichier⋅s odt voulus, puis : 
1. Ouvrez le fichiez odt 
2. Mettez à jour si nécessaire l'index (sommaire)
3. Mettez à jour le tableau de contribution
4. Générez un document PDF de cette version
